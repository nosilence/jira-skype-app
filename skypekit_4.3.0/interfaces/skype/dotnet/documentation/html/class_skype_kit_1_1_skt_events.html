SktEvents Class Reference

This class manages events and property updates for all the other classes. It should not be instantiated directly. The sole instance of this class - events field of your SktSkype object, can be used to assign custom callbacks to events and property updates. Description of class events and corresponding argument structs can be found under documentation of each individual class - that has any events - under the "Detailed Description" section of that class. More...

Inherits SkypeKit::SktEventsBase.

Detailed Description

This class manages events and property updates for all the other classes. It should not be instantiated directly. The sole instance of this class - events field of your SktSkype object, can be used to assign custom callbacks to events and property updates. Description of class events and corresponding argument structs can be found under documentation of each individual class - that has any events - under the "Detailed Description" section of that class.

NB! In order to prevent the entire content of this class from hopelessly cluttering up the main class index in the reference manual, documentation of all these classes has been excluded.
